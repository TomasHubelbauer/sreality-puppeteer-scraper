# SReality Puppeteer Scraper

Scrapes SReality.cs using Puppeteer and reports new and updated posts for a given search.

## Running

```sh
cd src
nvm use
yarn flat help
yarn house help
# Or `nodemon` for file change watching
node --experimental-modules index.mjs help
```
